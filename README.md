# MU-Bioinformática
Presentación de las actividades realizadas durante el Máster Universitario de Bioniformática, organizadas en carpetas para cada asignatura.

## Carpetas y contenidos
### Secuenciación y ómicas
- Proyecto QIIME2e: Análisis de datos metagenómicos con QIIME2. Archivos adicionales y script de código empleado para realizar análisis metagenómico.

### Introducción a la Programación Científica
- Fundamentos de Bash: Comandos básicos de Bash. Manejo de archivos, directorios y scripts.
