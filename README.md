# Proyecto-QIIME2-Atacama
Análisis de datos metagenómicos con QIIME2
Archivos adicionales y script de código empleado para realizar análisis metagenómico.

### Carpetas

#### Imágenes
Capturas de pantalla con resultados relevantes

#### qiime2-atacama
Directorio creado durante la ejecución del codigo con los archivos generados

#### Tablas exportadas BLAST
Tablas breves exportadas de BLAST para hacer la comparación taxonómica con los resultados de qiime2 + greengenes
