# MU-Bioinformática
Presentación de las actividades realizadas durante el Máster Universitario de Bioniformática, organizadas en carpetas para cada asignatura.

## Carpetas y contenidos
### Secuenciación y ómicas
#### Proyecto QIIME2e 
Análisis de datos metagenómicos con QIIME2. Archivos adicionales y script de código empleado para realizar análisis metagenómico.
##### Carpetas
- Imágenes: Capturas de pantalla con resultados relevantes
- qiime2-atacama: Directorio creado durante la ejecución del codigo con los archivos generados
- Tablas exportadas BLAST: Tablas breves exportadas de BLAST para hacer la comparación taxonómica con los resultados de qiime2 + greengenes
